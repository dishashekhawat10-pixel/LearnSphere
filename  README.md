#  LearnSphere – Study Resource Finder

##  Project Description

LearnSphere is a web application that helps users discover and explore study resources such as courses, tutorials, and learning materials. It provides an easy way to search, filter, and organize educational content using data from public APIs.

---

##  Planned Features

*  Search study resources by keywords
*  Filter resources by category, subject, or difficulty
*  Sort results based on relevance or alphabetical order
*  Save favorite resources
*  Responsive design for all devices

---

##  API to be Used

A public API will be used to fetch real-time educational resources (e.g., courses, tutorials, or study materials).

---

##  Technologies

* HTML
* CSS
* JavaScript (ES6)
* Fetch API

---

##  How to Run

1. Clone the repository
2. Open the project folder
3. Run `index.html` in your browser

---

##  Objective

To build a responsive web application that demonstrates API integration, use of JavaScript array methods (map, filter, sort), and dynamic UI rendering.
